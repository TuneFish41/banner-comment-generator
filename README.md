# banner-comment-generator


This script generates a banner comment.

Use from command line

```
>>> ./generate_comment.py
>>> Enter string: Config
>>> Enter comment: The user can override and set variables in config.cfg

# .--Config---------------------------------------------------------------.

# |                                             |___/                     |

# '-----------------------------------------------------------------------'
```
